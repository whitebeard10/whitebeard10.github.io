Sign Language Detection

Python | AI | ML | CNN

In this project, we developed a model using Convolutional Neural Network (CNN) to detect sign language and convert it into English alphabets. We collected a dataset of hand sign images and preprocessed them using OpenCV to apply transformations like black and white conversion, Gaussian blur, and binary inversion. Two different sequential models were created, and the second model included dropout to prevent overfitting, achieving an accuracy of 99.18%. The model was trained using CNN layers and compiled using the Adam optimizer algorithm with categorical cross-entropy as the loss function. We plotted graphs of loss and accuracy versus epochs to check for overfitting. The saved model was then deployed on a web app using Flask, and HTML with bootstrap templates was used for styling the pages. This project aimed to enhance communication for people who don't understand sign language and achieved high accuracy in detecting hand signs.
